# 2025年の振り返り

去年は後半が忙しすぎたので、何もできなかった1年の印象があったけど思い返してみたら色々イベントがあった。

体調
- 四十肩で肩が上がらなくなる
  - 半年ぐらい整体に通ってやっと治った
- 筋力の衰え
  - ボルダリングしてないので痩せ細ってる

仕事
- 少し仕事を増やした
- AI使いまくりで作業の効率がかなり上がった
  - でもコアの部分にはまだ全然使えてない
- 1件単発で大きめの仕事を受けた
- 入ってくるお金は増えたけど所得は減る見込みらしい

スト6
- ケンはハイマスター止まり
- Act8から豪鬼とキャミィを始めて、両方マスターになった
- 豪鬼はずっとMR1300付近で推移 => 11月ぐらいからMR1400~1500で安定するようになった

フォートナイト
- 被弾後の逃げ方と体力回復のためのボックスの広げ方が上手くなった
- マッチはほとんどやらなくなったけど地味に上達してる

不動産
- 少しずつ育ってる
- 今までで一番高い物件を買った
  - 多分無理だろうなと思ったけど出せる額で指値を入れたら通った
  - ひとまず順調
- 本厚木のアパートがとうとう満室
- 船橋の戸建が家賃滞納 => 退去 => リフォーム => 入居
  - 日本政策金融公庫からリフォーム費用を借りてみた
  - キャッシュフローは厳しいけど返済が早くてこれはこれであり

囲碁
- iOSの最強囲碁DLでCPUの3級にずっと勝てなかったが、突然勝てるようになってきた
- 布石の理屈が少しわかるようになったかも

旅行
- 今年は節約しようと思っていたので控えてたつもりだったけど全然行ってた
- フィンランド
  - サウナ巡り
    - 池には飛び込めなかった
  - マイレア邸見学
    - 建築家の人と一緒に行って色々教えてもらいながら見た
    - ガイドもわかりやすくて良かった
    - 行く前はただの家を見て面白いか分からなかったけど、実際見てみるとかなり興味深かった
  - 食べ物が失礼ながら意外と美味しかった
- 越後湯沢
  - いつもの将棋合宿
- 名古屋
  - 仕事関係で何回か行った
  - 毎回何かしら美味しいものをご馳走になってる
  - まだエビフライを食べてない
- 草津温泉
  - 硫黄の匂いがすごかった
  - 宿が渋すぎた
- 福井
  - 友人が目的があって行くというのでついて行った
  - 永平寺と恐竜博物館に行った
  - 恐竜博物館は建築がすごかった
  - 永平寺の近代化された供養受付が良かった
- シンガポール
  - 藤井-伊藤戦を見に行った
  - スロットで3000SGD負け
  - 一緒に行った友人と徹夜でブラックジャックを打ったのが楽しかった
- インドネシア
  - シンガポールから寄り道
  - 田舎の街や博物館が良かった
- 嬬恋村
  - 不動産関係で行って、温泉旅館に泊まった
  - 旅館のご飯が美味しかった
- 札幌
  - 父に会いに北海道に行って、札幌にも泊まった
  - ジンギスカンを食べて蟹を買ってきた
- ドバイ
  - 不動産を見がてら観光
  - 不動産は良かった
    - 現金があれば東京のマンションを買うよりドバイの方が楽しい
  - 観光地が全力で観光に力を入れてるのが良かった
  - アブダビのルーブル美術館が見応えがあった
- 石垣島
  - いつもの将棋大会見学
